# Anime Style Gen for OpenClaw

A professional image-to-image (I2I) conversion skill for OpenClaw, powered by Gemini 3 Pro Image via DMXAPI.

## Features
- **Zero-loss I2I**: Direct image-to-image conversion preserving composition and expressions.
- **Multiple Styles**: Pixar 3D, Studio Ghibli, Makoto Shinkai, and Traditional Ink.
- **Card Mode**: Automatic framing into decorative trading cards and holographic "holo" cards.
- **Seamless Integration**: One-click generation and delivery to Feishu/Discord/Telegram.

## How to use
Just send a photo and say "Transform into Ghibli style card".
