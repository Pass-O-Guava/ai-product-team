# 标准化工作流

> 团队协作的标准流程和最佳实践

## 流程列表

### 1. 任务接单流程
WY 下达需求 → @pm 分析 → 拆解任务 → 调度子智能体 → 汇总交付

### 2. 产品迭代流程
想法 → @pomelo PRD → @researcher 调研 → @designer 原型 → @engineer 实现 → @reviewer 验收 → 交付

### 3. Skills 沉淀流程
任务完成 → 识别可复用能力 → 写成 SKILL.md → 更新 /team/skills/

### 4. GitHub 备份流程
每次 commit → 自动推送到 GitHub → @pm 通知 WY
