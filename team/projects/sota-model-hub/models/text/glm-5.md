## 核心能力

- **MIT 许可证开源**：智谱AI首次以 MIT 许可证开源 Frontier 级大模型
- **200K 超长上下文**：支持 200K token 上下文窗口
- **复杂系统工程 + Agent 任务**：针对复杂系统级工程和长时序 Agent 工作流优化
- **Benchmark 对标 Claude Opus 4.6**：Artificial Analysis 显示综合能力接近 Claude Opus 4.6 水平
- **配套 GLM-5V-Turbo**：同时发布 GLM-5V-Turbo 视觉编程版本（闭源 API）

中国大厂首次以 MIT 许可证开源 Frontier 级模型，意义重大。综合能力对标 GPT-4.5/Claude Opus 4.6 级别，同时配套开源视觉编程版本，在开源社区具有标杆意义。
