## 核心能力

- **全规格覆盖**：从 0.5B 到 72B，涵盖从手机到服务器的全面部署需求
- **128K 上下文**：支持 128K token 超长上下文
- **强代码能力**：Qwen2.5-Coder 系列代码能力突出
- **多语言**：支持中英等数十种语言
- **超强指令跟随**：IFEFF、ITR 等对齐技术提升指令遵循能力
- **长文本生成**：生成最长 8K token

## 备注

Qwen2.5 是全球最具影响力的开源 LLM 系列之一，七种规格覆盖所有部署场景。0.5B 版本可运行在手机和树莓派上，72B 版本达到顶级性能，是开源 LLM 民主化的标杆产品。
