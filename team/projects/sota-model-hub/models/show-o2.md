# Show-o2

> **数据可靠度：A**（NeurIPS 2025 & arXiv 2506.15564）

| 字段 | 内容 |
|------|------|
| **模型名称** | Show-o2 |
| **发布方** | Showlab（NUS 新加坡国立大学） |
| **参数量级** | 1.5B / 7B |
| **支持的模态** | 文本 + 图像理解 + 图像生成（统一框架） |
| **许可协议** | Apache 2.0 |

## 核心能力
自回归 + 流匹配混合：Show-o2 融合 AR 和 Flow Matching 两种生成范式；7B 版本超越 14B 竞品（更低训练数据）；统一理解与生成；ICLR 2024 Show-o 的改进版。

## 适用场景
统一多模态理解+生成、低训练成本多模态研究、图文创作
