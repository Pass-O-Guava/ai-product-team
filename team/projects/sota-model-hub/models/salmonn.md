# SALMONN

> **数据可靠度：A**（ByteDance 官方 & ICLR 2024 论文）

| 字段 | 内容 |
|------|------|
| **模型名称** | SALMONN（Speech Audio Language Music Open Neural Network） |
| **发布方** | 字节跳动（ByteDance） |
| **参数量级** | 7B / 13B |
| **支持的模态** | 音频（语音+音乐+音效）+ 文本 |
| **许可协议** | CC BY-NC-SA 4.0（非商业） |
| **HuggingFace** | https://huggingface.co/bytedance/salmonn |
| **魔搭 ModelScope** | 暂无 |
| **发布年份/版本** | 2024年（ICLR 2024） |

## 核心能力
通用听觉 LLM：同时支持语音、音乐、自然音效理解；语音增强；音乐情感分析；音频事件检测。SALMONN-omni（2025年）新增全双工语音 LLM，无需音频编解码器。

## 适用场景
音视频内容分析、语音增强、音乐推荐、环境声音监测、视频理解音频通道
