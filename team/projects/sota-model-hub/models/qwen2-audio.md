# Qwen2-Audio

> **数据可靠度：A**（Qwen 官方博客 & arXiv 2407.10759）

## 基本信息

| 字段 | 内容 |
|------|------|
| **模型名称** | Qwen2-Audio / Qwen-Audio-Chat |
| **发布方** | 阿里巴巴通义实验室（Alibaba Cloud） |
| **参数量级** | 7B |
| **支持的模态** | 音频（语音/音乐/音效）+ 文本 |
| **许可协议** | Tongyi Qianwen License |
| **HuggingFace** | https://huggingface.co/Qwen/Qwen2-Audio-7B |
| **魔搭 ModelScope** | https://modelscope.cn/models/Qwen/Qwen2-Audio |
| **发布年份/版本** | 2024年8月发布 |

## 核心能力

- 全类型音频理解：语音、音乐、自然音效
- 语音聊天 & 多语言（8+语言/方言）
- 自然语言提示驱动音频分析
- 音频问答、摘要、情感分析

## 适用场景

语音助手、音乐理解、音视频字幕、音频档案分析
