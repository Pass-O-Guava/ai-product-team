## 核心能力描述

GLM-4.6V 是智谱 AI 推出的旗舰级视觉语言模型，主打"原生工具调用"：

- **原生多模态工具调用（Native Tool Calling）**：业界首创，将视觉感知与工具调用能力原生融合，模型可直接调用外部工具（如浏览器、计算器）处理视觉任务
- **131K 超长上下文**：支持 131K token 的多模态上下文，可一次性处理大量图文内容
- **前端自动化**：支持 GUI/截图 自动化操作，可作为视觉 Agent 驱动网页和桌面应用
- **大规模图文混合内容理解**：长文档、报告、网页截图等复杂内容的端到端理解
- **API 价格亲民**：智谱同步大幅降低 API 价格，GLM-4.6V 性价比极高
- **交错图文生成**：支持图文交错输出的复杂内容创作
