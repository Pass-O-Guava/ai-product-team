# SOTA Radar · 模型知识飞轮

> 基于 OpenClaw 多智能体架构的 SOTA 模型知识库

## 项目结构

```
ai-product-team/
├── frontend/          # 前端（React + TypeScript + Vite）
├── backend/          # 后端（Python FastAPI）
└── .github/          # CI/CD 工作流
```

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | React + TypeScript + Vite |
| 后端 | Python FastAPI + Uvicorn |
| 知识存储 | Markdown + JSON 文件 |
| 部署 | MiniMax Space |
| 团队协作 | OpenClaw 多智能体 |
| CI/CD | GitHub Actions |
